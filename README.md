#  Margadarshan – Career Guidance Platform

## 📌 Overview

**Margadarshan** is a career guidance platform designed to help students explore career options, get guidance, and make better decisions about their future.
It provides structured information, resources, and support for different career paths.

---

## 🎯 Aim

To build a user-friendly platform that helps students:

* Discover career opportunities
* Get proper guidance
* Make informed decisions about their future

---

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Python (Flask / FastAPI)
* REST APIs

### Database

* Supabase / PostgreSQL

---

## ⚙️ Features

* 🔐 User Authentication (Login/Signup)
* 📊 Career Recommendations
* 📚 Career Information & Resources
* 🧑‍💼 Guidance for students
* 🌐 Responsive UI
